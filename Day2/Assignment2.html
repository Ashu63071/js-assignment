Javascript some Array Methods

.toString():Convert all the elements of the array into string

array.splice([startpos],[no.of elements to remove],[number of items to insert]): remove the elements and it returns the deleted items.

array.sort(): Sorts the array.

array,slice(): Slice a portion of the array.

For Example: Syntax are:

let array=['hii','cat','dog', 5676, 2337.7, 357556n, true];
console.log(array.toString());
console.log(array.toLocaleString());
console.log(array.splice(2,2,"hey","hola"));
console.log(array);
console.log(array.sort());
console.log(array.slice(0,4));

Javascript some String methods

.substring: returns a part of the string , no negative values are allowed.

.toLocaleLowerCase: returns the whole string in local lower case.

.toUpperCase: returns the whole string into upper case.

.search(): search a string and returns the index position.

.substr(index,length): returns the subpart from the index position till the no. of character(whose length is given in second part.

.replace("replacable string","replaced string"): returns the string with the replaced parts.'//g' replaces globally, while '//i' replaces insensitive of case.

.trim(): Remove extra spaces in the string and returns the whole string.

.split('str'): converts the string into array by spliting the whole string into substrings from the str.

For example syntaxes are:

let line = " Beta bought some butter, but butter bought by Beta was bitter. So Beta bought some more better butter to make the bitter butter better "
console.log(line);
console.log(line.substring(63,101));
console.log(line.toLocaleLowerCase());
console.log(line.toUpperCase());
console.log(line.search('butter'));
console.log(line.substr(17,6));
console.log(line.replace(/bitter/g,"stale"));
console.log(line.trim());
console.log(line.split('butter'));
